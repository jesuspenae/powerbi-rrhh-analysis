# Workforce Analytics Dashboard (HR) - Power BI

This project features a comprehensive Human Resources dashboard designed to monitor and analyze key performance indicators (KPIs) for a workforce of 3,520 employees.

## 📊 Business Case
The objective was to create a centralized view of the company's demographics and operational status, allowing HR managers to make data-driven decisions regarding staff distribution and diversity.

## 🚀 Technical Highlights
* **DAX Implementation:** Developed custom measures to transform categorical text-based IDs into precise quantitative metrics for "Active" and "Inactive" status tracking.
* **Data Modeling:** Integrated employee datasets to enable seamless filtering by department, country, and education level.
* **UX/UI Design:** Customized the visual interface with a clean aesthetic, ensuring all high-value cards (Total Employees, Active/Inactive) display exact figures with thousand separators for maximum clarity.

## 🔍 Key Insights
* **Global Footprint:** Visual mapping of staff distribution across Spain, France, Portugal, and Switzerland.
* **Demographics:** Real-time breakdown of gender diversity, education levels (Graduate, Postgraduate, Undergraduate), and contract types (Full-time, Part-time).
* **Workforce Status:** Clear identification of active vs. inactive personnel to monitor turnover rates.

## 📁 Repository Contents
